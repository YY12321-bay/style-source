## 🎨 视觉设定

### 主色调
| 色块 | 用途 | 色值 |
|------|------|------|
| ⚪ #E0E0E0 | 主体亮色/高光 | RGB(224,224,224) |
| ⚫ #404040 | 主体深色/轮廓 | RGB(64,64,64) |
| ⬛ #202020 | 深色细节 | RGB(32,32,32) |
| 🔵 #C0E0E0 | 冷色点缀/屏幕光 | RGB(192,224,224) |
| 🔵 #C0C0E0 | 蓝紫调高光 | RGB(192,192,224) |

### 形象特征
- **整体**：较高的竖向主体，灰色调和冷色点缀
- **体态**：纵向较长（几乎占满画面高度），横向较窄
- **风格**：简约现代，有电视/屏幕相关的元素感
- **主体**：灰白基调，带冷蓝色光效或装饰
- **视觉重心**：画面居中，竖直拉长的轮廓

## 🧠 角色性格

| 维度 | 设定 |
|------|------|
| **性格关键词** | 资讯、娱乐、陪伴、多彩 |
| **品牌联想** | 电视节目、影视剧、新闻直播 |
| **标志元素** | 电视荧幕、遥控器、频道、播放按钮 |
| **氛围** | 家庭客厅、演播厅、巨幕影院 |

## 📸 常用姿态
1. **荧幕展示** — 屏幕显示丰富多彩的内容画面
2. **直播主持** — 演播厅氛围，聚光灯效果
3. **家庭陪伴** — 在客厅沙发旁，温馨氛围
4. **影视推荐** — 周围环绕电影胶片/播放列表元素

## 🎬 场景适应性
